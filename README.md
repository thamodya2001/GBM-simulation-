# 📈 Geometric Brownian Motion (GBM) Simulation Project

This project focuses on simulating stochastic processes using **Geometric Brownian Motion (GBM)**, a widely used model in financial mathematics for representing the dynamic behavior of asset prices over time. The simulation captures both deterministic trends (drift) and random fluctuations (volatility), providing a realistic framework for modeling asset price movements under uncertainty.



This work was carried out by me as part of an assignment for a university course, demonstrating practical implementation of stochastic modeling concepts and simulation techniques.
